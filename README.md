# SwipeSweep（iOS 照片管理应用）

SwipeSweep 是一款 iOS 照片管理/清理应用，通过直观的滑动手势快速完成“删除、保留、收藏、批量操作”等整理流程，帮助你更轻松地清理相册与释放空间。
